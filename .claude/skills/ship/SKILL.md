## Ship
1. Run `npm run build` and fix any errors
2. Run `git status` and show the COMPLETE list of changed/untracked files — flag anything unexpected
3. Stage ALL changes with `git add -A`
4. Run `git diff --cached --stat` to confirm the full set of staged files
5. Generate a concise commit message from the diff
6. `git commit` and `git push`
7. Report the full list of files that were pushed and confirm nothing was left behind
